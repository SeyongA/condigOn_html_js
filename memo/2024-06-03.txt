박스모델
평면 - 너비(width), 높이(height)
가운데 정렬(margin)
right left - auto


가로 정렬 (div)
display : inline-block <- 가로정렬
display : block <- 세로정렬 (기본값)

마진중첩 : 수직 배치일 경우에 margin - top, bottom은
         마진이 중첩될 경우 마진이 큰쪽으로 설정된다.

선택자
1. 태그 선택자
2. id 선택자 - #으로 시작
3. class 선택자 - .(점)으로 시작

스타일 적용
1. 내부 스타일 <style>~</style>
2. 외부 스타일 <link>로 연결 .css
