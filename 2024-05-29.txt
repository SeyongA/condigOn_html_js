* cmd 명령어

	cd - 디렉터리 이동(폴더 이동)
	dir - 폴더, 파일 목록보기
	c:/ (c의 루트)

* git 명령어
	git  계정 설정
	
	git config --global user.name "name"
	git config user.name
	git config --global user.email "email"
	git config user.email

* 해당 폴더 이동
	git init : 깃 생성
	git add : 깃에 추가
	git commit -m "test" : 커밋 실행
	git remote add origin 주소 :  경로 설정
	git push -u origin master
	
	git status : 상태 확인
